Interface ITable

A table is a data structure made up of rows and columns. In a table, the elements within a column are always of the same data type; the data types between different columns can vary. For example, the first column in a table can be categorical, the second can be integers, the third can be IDs, etc.

If your columns are of the same type, use Matrix instead.
